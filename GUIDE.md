## Understanding TruthBot's Verdicts

TruthBot gives one of these verdicts for each claim:

| Verdict | Meaning |
|---|---|
| ✅ **TRUE** | The claim is accurate based on verified sources |
| ❌ **FALSE** | The claim is demonstrably incorrect |
| ⚠️ **MISLEADING** | Contains true elements but creates a false impression |
| 🔴 **MOSTLY FALSE** | Mostly wrong with minor accurate elements |
| 🕐 **OUTDATED** | Was true at some point but no longer accurate |
| 🔍 **MISSING CONTEXT** | True but omits critical information |
| 🔄 **OUT OF CONTEXT** | Real content used in the wrong context |
| ❓ **UNVERIFIED** | Not enough evidence to determine — be cautious |
| 🤖 **AI-GENERATED** | Image or video was created by AI |

### Confidence Levels

Every verdict comes with a confidence level:

- 🟢 **High confidence** — Multiple trusted sources agree
- 🟡 **Medium confidence** — Some sources found, take with a pinch of salt
- 🔴 **Low confidence** — Limited sources available, be extra cautious

## Tips for Best Results

1. **Forward the full message** — more context = better fact-checking
2. **One thing at a time** — send one claim, image, or link per message
3. **Be specific** — "Is X true?" works better than vague questions
4. **Check images too** — many fake claims spread as screenshots or AI images
5. **Don't just trust the forward count** — a message forwarded 1000 times can still be fake

## How TruthBot Works (For the Curious)

TruthBot uses a **4-layer source trust system** to verify claims:

1. **Fact-Check Organizations** (highest trust) — BOOM Live, Alt News, Snopes, AFP Fact Check, etc.
2. **Official Sources** — RBI, WHO, PIB, ICMR, NASA, government websites
3. **Established News Outlets** — NDTV, The Hindu, Reuters, BBC, etc.
4. **General Web Search** — Used for context only, never cited directly

TruthBot also uses AI to:
- Extract specific claims from long messages
- Detect AI-generated images and deepfake videos
- Read text from screenshots (OCR)

**TruthBot never makes things up.** If it can't find evidence, it says "UNVERIFIED" rather than guessing.
